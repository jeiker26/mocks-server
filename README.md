# mocks-server
Simple mocks server

## How to use
Click on this repository and add as many sections / micros that you want, as it appears in the `generate.js` file, then:
```js
npm run generate
```
You can see how a collection of data has been created in the `database.json` file.

## Run server
```js
npm run start
```
